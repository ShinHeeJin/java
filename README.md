# Java 100 Examples - 기초부터 고급까지

이 프로젝트는 Java를 배우기 위한 100가지 예제를 포함합니다. 기초 개념부터 고급 주제까지 체계적으로 구성되어 있습니다.

## 📚 학습 가이드

| 카테고리 | 개수 | 설명 |
|---------|------|------|
| **기초 (Basics)** | 15개 | 변수, 연산자, 제어문, 배열 |
| **객체지향 (OOP)** | 20개 | 클래스, 상속, 다형성, 인터페이스 |
| **컬렉션 (Collections)** | 15개 | List, Set, Map, 정렬 |
| **문자열과 정규표현식** | 10개 | String, StringBuilder, Regex |
| **파일 I/O** | 10개 | 파일 읽기/쓰기, 직렬화 |
| **예외 처리** | 10개 | try-catch, 사용자 정의 예외 |
| **제네릭** | 10개 | 제네릭 클래스, 메서드, 와일드카드 |
| **함수형 프로그래밍** | 15개 | Lambda, Stream, 메서드 참조 |

## 🚀 실행 방법

```bash
# 기초 예제 실행
javac basics/Example01_Variables.java
java basics.Example01_Variables

# 모든 파일 컴파일
find . -name "*.java" -type f -exec javac {} \;
```
